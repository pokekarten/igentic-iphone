# iGentic Brand Assets

Last updated: 2026-06-12

This folder contains initial editable SVG assets for the iGentic open-source project identity.

## Assets

| File | Use |
| --- | --- |
| `logo-mark.svg` | Icon-only mark for avatars, app icon experiments and small placements |
| `logo-lockup.svg` | README/website header lockup with mark, wordmark and tagline |
| `social-card.svg` | GitHub/social preview card candidate |

## Design concept

The mark combines:

- a lowercase `i` for identity, user and local device trust,
- an open uppercase `G` for guardrails, governance and agent loop,
- a subtle policy/delegation line,
- calm silver-blue accents.

## Usage rules

Follow:

- `docs/brand/BRAND.md`
- `docs/brand/DESIGN_SYSTEM.md`
- `docs/brand/LOGO_BRIEF.md`
- `docs/brand/LOGO_USAGE.md`

## Accessibility rules

Every SVG should include:

- `<title>`
- `<desc>`
- `role="img"`
- meaningful alt text wherever embedded in markdown or HTML.

## Review checklist

Before treating an asset as official, check:

- readable at small sizes,
- works in monochrome or has a monochrome variant,
- does not look like an Apple-owned mark,
- does not use an iPhone outline, Dynamic Island, camera bump or Apple logo,
- communicates control, privacy and trust rather than generic AI hype,
- can be understood by contributors from the README.

## Current status

These are initial assets, not final brand-lock assets. They are intended to make the repository look coherent while leaving room for community design feedback.
