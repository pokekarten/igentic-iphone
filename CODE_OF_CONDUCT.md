# Code of Conduct

## Our pledge

We want iGentic iPhone to be a respectful, constructive and privacy-conscious open-source project.

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, education, nationality, personal appearance, race, religion, sexual identity and orientation, or technology background.

## Expected behavior

Examples of behavior that contributes to a positive environment:

- being respectful and constructive,
- assuming good intent while still reviewing carefully,
- giving clear technical feedback,
- documenting risks and tradeoffs honestly,
- respecting privacy and security boundaries,
- avoiding pressure to share personal data or credentials.

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment, insults or personal attacks,
- sexualized language or imagery,
- trolling or sustained disruption,
- publishing private information without explicit permission,
- requesting or sharing secrets, credentials or private user data,
- intentionally bypassing privacy, approval or audit safeguards,
- presenting speculative model or device claims as verified facts.

## Privacy-specific rule

This project is about privacy-first AI. Contributors must not ask others to upload private data, production secrets, health information, financial information, messages, contacts, calendars or identity material into public issues or pull requests.

Use synthetic data for examples and tests.

## Enforcement

Project maintainers may remove comments, commits, code, issues or pull requests that violate this Code of Conduct. Maintainers may also temporarily or permanently restrict participation for behavior they deem inappropriate, threatening, offensive or harmful.

## Reporting

If you see unacceptable behavior, report it privately through the repository owner or the security contact described in `SECURITY.md` when the issue involves privacy or security.

## Attribution

This Code of Conduct is inspired by common open-source community standards and adapted for a privacy-first AI runtime research project.
