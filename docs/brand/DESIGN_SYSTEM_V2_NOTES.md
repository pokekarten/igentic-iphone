## Visual direction

The v2 direction is intentionally simpler than the first exploratory mark.

Design priorities:

1. recognizable at 64 px,
2. centered inside circular avatar crops,
3. editable as SVG,
4. strong blue control ring,
5. calm graphite identity point,
6. no tiny decorative details in profile assets.

## Review rule

If a proposed logo needs a tagline to explain it, it is not strong enough for an avatar. The avatar should work with the mark alone.
