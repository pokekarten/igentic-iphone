# iGentic Logo Brief

Last updated: 2026-06-12

## Objective

Create a distinctive logo system for **iGentic** that can work as:

- GitHub repository identity,
- app icon candidate,
- social profile picture,
- website mark,
- community project badge,
- future product/IP foundation.

## Primary concept

The logo should combine a lowercase `i` and an uppercase `G` into a single minimal control mark.

The `i` represents:

- identity,
- the user,
- the trusted local device,
- explicit consent.

The `G` represents:

- guardrails,
- governance,
- agent loop,
- controlled delegation,
- policy boundaries.

The ring should be slightly open. A fully closed ring can feel locked or sealed; an open ring communicates controlled flow.

## What the logo should say

> The human is at the center. Intelligence is allowed to operate only inside visible policy boundaries.

## Visual requirements

- Minimal, geometric, rounded.
- Strong silhouette at small sizes.
- Works in monochrome.
- Works as a 1024x1024 app icon.
- Works as a small GitHub/social avatar.
- Uses subtle silver-blue gradients only in premium versions.
- Has a flat single-color variant for docs, CI badges and diagrams.

## Avoid

- Apple logo or Apple-like fruit shapes.
- iPhone outlines, camera bumps, Dynamic Island shapes or product silhouettes.
- Siri-like glowing orbs.
- Generic robot heads.
- Chat bubbles.
- Overly complex neural network dots.
- Neon cyberpunk colors.
- Anything that implies Apple endorsement.

## Logo variants needed

1. `logo-mark.svg` — icon-only `iG` mark.
2. `logo-wordmark.svg` — `iGentic` text only.
3. `logo-lockup.svg` — mark + wordmark.
4. `app-icon.svg` — rounded-square app icon mock.
5. `social-card.svg` — GitHub/social preview card.
6. `monochrome.svg` — one-color version.

## Color direction

Primary:

- graphite text,
- frosted silver mark,
- air-blue accent.

Secondary:

- white background,
- mist-blue glow,
- dark-mode graphite/navy surfaces only when needed.

## Wordmark direction

The wordmark should be clean and neutral, using system-font proportions. It should not depend on Apple proprietary typography. The `i` should stay lowercase and the `G` should stay uppercase: `iGentic`.

## App icon direction

- Rounded square.
- White or mist background.
- Centered `iG` mark.
- Very soft inner shadow or border.
- No device outline.
- No Apple-style hardware reference.

## Test checklist

A logo candidate passes when:

- it remains readable at 32x32,
- it works in black and white,
- it does not look like an Apple-owned mark,
- it communicates control rather than generic AI,
- it can be used by community members without confusion,
- it can become a long-term project identity.

## Prompt for future logo generation

Create a minimalist, premium open-source AI runtime logo for `iGentic`. Combine a lowercase `i` and an uppercase `G` into a single rounded control-ring mark. The `i` represents user identity and local device trust; the open `G` represents guardrails, policy boundaries and controlled delegation. Use a calm graphite, frosted silver and subtle air-blue gradient. Make it clean, precise, high-trust and future-facing. Do not include an Apple logo, iPhone silhouette, camera bump, chat bubble, robot head or Siri-like orb. The result must work as an app icon, GitHub avatar and open-source community mark.
