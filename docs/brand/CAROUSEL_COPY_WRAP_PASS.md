# Carousel Copy Wrap Pass

Last updated: 2026-06-14

## Purpose

Validate short, wrapped cover copy for the first iGentic social carousel topics before the carousel template is promoted from candidate toward default.

This pass is visual/documentation-only. It does not change Swift runtime code, workflows, networking, providers, signing files, private data or Pokekartenkiste files.

## Source template

`assets/social/instagram-carousel-template-v1.svg`

The current template uses a 1080 x 1350 frame with a single-line subtitle. The subtitle line should be replaced with two shorter lines for realistic public copy.
